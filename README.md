#  SQL Mastery Roadmap: From Foundations to Advanced Problem Solving

This repository documents my systematic journey to mastering SQL. It covers everything from core syntax to complex analytical queries, designed to handle large-scale data challenges in various roles such as Data Analysis, Business Intelligence, and Data Engineering.

## 📈 Learning Roadmap

### **Module 1: Advanced SQL & Analytical Thinking**
* **Day 1: Mastering Window Functions** ✅
    * *Core Skills:* Data partitioning, ranking logic, and trend analysis using professional window frames.
    * *Key Syntax:* `OVER()`, `PARTITION BY`, `ROWS/RANGE BETWEEN`, `LEAD/LAG`.
    * *Deliverable:* [Day1_Window_Functions.sql](./Day1_Window_Functions.sql)

* **Day 2: Structural Query Design (CTEs & Subqueries)** ⏳
    * *Goal:* Writing clean, modular, and maintainable code for complex business logic.

* **Day 3: Relational Data Modeling & Self-Joins** ⏳
    * *Goal:* Navigating hierarchical data and complex table relationships.

## 💡 Why This Repository?
The goal of this roadmap is not just to learn syntax, but to develop an **Analytical Mindset**. 
- **Scalability:** Writing queries that perform well on large datasets.
- **Readability:** Using CTEs and professional commenting to ensure code can be audited by teammates.
- **Logic over Syntax:** Understanding *why* a specific Window Frame (like `RANGE INTERVAL`) is chosen over others for specific business cases.
